Встраивание плеера Kinescope во всплывающем окне (pop-up) блоки T868 и T1093

Всплывающие окна (pop-up) могут появляться в зависимости от поведения пользователя: при клике на ссылку, при скролле, по времени или при закрытии страницы.
Содержимым такого блока может быть плеер Kinescope. Для стабильной работы плеера в этом случае используется скрипт.

Плеер Kinescope в блоке T868

Pop-up c видео 9:16 Pop-up c видео 16:9

Примеры встраивания плеера Kinescope

С превью с сохранением соотношения сторон 16:9

<script src="https://kinescope.dev/0SmBbQxsRvAf1Z458SiP7v/popup.js?aspect_ratio=16/9"></script>

С превью фиксированного размера 200x200

<script src="https://kinescope.dev/0SmBbQxsRvAf1Z458SiP7v/popup.js?width=200&height=200"></script>

С превью заполняющим родительский блок

<div style="width:100%; height:160px;">
<script src="https://kinescope.dev/0SmBbQxsRvAf1Z458SiP7v/popup.js"></script>
</div>

Без превью в ссылке

<a> Click to open popup
<script src="https://kinescope.dev/0SmBbQxsRvAf1Z458SiP7v/popup.js?inline=1"></script>
</a>
